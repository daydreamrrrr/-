Лабораторна робота №20

Робота в мережі

В.3
Створити програму для пересилання файлів по мережі.

Хід роботи

Під час лабораторної роботи я створила програму-клієнт та програму-сервер з використанням інструментів Swing для пересилання файлів по мережі. В клієнті є можливість вибрати файл для пересилання, в сервері цей файл зберігається та його ім'я виводиться на JFrame.
